Setup — what you actually do

In your KiX portal, go to Integrations → TikTok and paste two things:

# In your TikTok Ads Manager → Assets → Events → Pixel: TikTok Pixel ID: "C4FXX...your-pixel-id..." Events API Token: "a1b2c3d4...your-access-token..." # That's it. KiX maps its internal events to TikTok Standard Events automatically. # Test it: play a game on your own merchant page, then check TikTok Events Manager # under 'Test Events' tab — events appear within 30 seconds.

Want event-name overrides or custom-property mapping? Reach out to support — we'll JSON-spec it for you.

FAQ — what TikTok-savvy operators actually ask

Does this work for Spark Ads / Smart Performance Campaign optimisation?

Yes. The server-side Purchase event flows into TikTok's Smart Performance Campaign signals, so you can run conversion campaigns optimising for KiX-tracked transactions. We've validated this with TikTok Ads Manager v3.4+.

How does iOS 14+ (no IDFA) impact attribution?

The server-side Events API path is the iOS-14-resilient one. We send the user's phone/email (hashed SHA-256) as match_keys, which TikTok uses for cross-device match without IDFA. Match rate in our SG pilots: 78–84%.

Can I retarget KiX game-players who didn't redeem?

Yes. Use the CompleteRegistration custom audience (built from KiX events) as the source for a "registered-but-didn't-purchase" retarget campaign. We've seen merchants hit 4.2x ROAS on these.

What about Meta Pixel, GA4, Google Ads CAPI?

Same architecture: paste your Pixel/Tag ID in Integrations, KiX fires equivalent events. See /landing/connect.html for the full integration list.

Can I add custom properties (e.g. brand_id, outlet_id, game_id)?

Yes. Custom properties are pass-through by default for outlet_id, game_id, voucher_id, campaign_id. Add your own via kix.setEventContext({...}) in the SDK.

Does this respect Malaysian PDPA / Singapore PDPA / GDPR?

Yes. We hash all PII (phone, email) before sending to TikTok. Customer consent for ad-platform sharing is captured in the KiX game flow per market — see DPA.
